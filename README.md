# Tetris

Tetris（俄羅斯方塊）遊戲
使用 Java + Swing 實作的俄羅斯方塊遊戲（Tetris）

- 專案包含：
  - 核心遊戲邏輯
  - 模型
  - 鍵盤操作
  - UI

## 主要功能

- 可遊玩的俄羅斯方塊（七種 Tetromino：I, O, T, S, Z, J, L）
- 自動下落（每 500ms）與秒數計時（每秒更新）
- 可調整難度（`1` 簡單、`2` 普通、`3` 困難）
- 碰撞檢測與方塊鎖定（freeze）
- 消行偵測與數行得分：1 行 100、2 行 300、3 行 500、4 行 800
- 硬降（Hard Drop）與軟降（Soft Drop）
- 方塊旋轉（含碰撞檢查與還原）

## 快速上手（需求）

- Java 17 或以上 JDK

## 操作說明（鍵盤）

- ← / →：左右移動方塊
- ↑：旋轉方塊（順時針）
- ↓：軟降（加速下落）
- 空白鍵（Space）：硬降（立即落到最底並鎖定）
- P 或 Esc：暫停 / 恢復遊戲
- 1 / 2 / 3：切換難度（簡單 / 普通 / 困難）
